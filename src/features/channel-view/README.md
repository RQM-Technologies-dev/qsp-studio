# Channel View

> **Status: Planned — not yet implemented**

The Channel View will simulate how quaternionic signals propagate through physical
channels, demonstrating how rotation, attenuation, and dispersion affect the unified
geometric state.

## Planned capabilities

- Configurable channel model (AWGN, multipath, polarization rotation)
- Side-by-side input/output signal visualization
- BER and SNR displays for quaternionic vs. classical receivers
