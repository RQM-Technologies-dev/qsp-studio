# Signal Builder

> **Status: Planned — not yet implemented**

The Signal Builder will allow users to compose arbitrary quaternionic signals from
first principles: choosing basis functions, quaternion paths, and modulation strategies.

## Planned capabilities

- Define signal geometry using quaternion-valued basis functions
- Layer multiple components (carrier, envelope, modulation)
- Preview signal trajectories in the main 3D scene
- Export signal definitions as JSON for use in other Studio modules
