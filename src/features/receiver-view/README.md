# Receiver View

> **Status: Planned — not yet implemented**

The Receiver View will simulate quaternionic signal reception and detection,
showing how a quaternionic receiver can exploit full geometric state information
that classical receivers discard.

## Planned capabilities

- Matched filter and correlator receiver models
- Geometric state estimation from noisy observations
- Performance comparison: quaternionic vs. classical receiver
- Visualization of decision boundaries in 3D signal space
